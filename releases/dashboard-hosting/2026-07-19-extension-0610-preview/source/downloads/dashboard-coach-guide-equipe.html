CFSB Coach Command

Guide equipe - Dashboard Coach

Le dashboard sert a regrouper les priorites quotidiennes, clients, questionnaires, rebooking et pilotage coach dans un seul endroit.

1. Premiere connexion

  1. Ouvre le dashboard: https://cfsb-dashboard-coach-aa9a4.web.app.
  2. Connecte-toi avec Google ou avec courriel + mot de passe.
  3. Confirme que ton nom apparait dans le menu coach.
  4. Si tu vois le mauvais coach, choisis ton nom dans le selecteur.
  5. Commence par la page To-do.
Chaque coach peut consulter les autres coachs pilotes pour couvrir une absence ou aider l'equipe, mais les actions importantes restent journalisees.

2. Connexion Google ou mot de passe

Connexion Google Utilise cette option si ton compte Google est associe au courriel autorise.
Courriel + mot de passe Utilise cette option si tu n'as pas de compte Google ou si Google boucle sur mobile.

3. Routine quotidienne

  1. To-do: traite les missions concretes: programme, questionnaire, rebooking ou note coach.
  2. Clients: cherche un client, consulte sa fiche, ajoute une mission ou corrige une information.
  3. Questionnaires: lis les reponses, envoie un questionnaire ou gere les automatisations.
  4. Rebooking: confirme les seances a remettre, ajoute une seance manuelle si elle manque, puis ferme le dossier.
  5. Pilotage: prepare ou complete la rencontre hebdo avec les notes, IDS et indicateurs.

4. Installer l'extension CoachRx

  1. Dans le Guide du dashboard, telecharge le ZIP CoachRx Sync.
  2. Clic droit sur le ZIP, puis Extraire tout.
  3. Dans Chrome, ouvre chrome://extensions.
  4. Active le mode developpeur.
  5. Clique Charger l'extension non empaquetee.
  6. Selectionne le dossier extrait, pas le fichier ZIP.
  7. Dans le Guide, copie l'URL Apps Script et le secret, puis colle-les dans l'extension.
  8. Ouvre la page Clients CoachRx du bon coach et clique Mettre a jour CoachRx.

5. Verification apres sync CoachRx

  • Le nombre de clients recus doit ressembler a CoachRx.
  • Les signaux rouges ou jaunes CoachRx creent des missions programme.
  • Les clients sans signal restent dans Clients; ils ne doivent pas devenir des To-do.
  • Si un telephone manque, corriger la fiche ou la source avant de tester GHL/questionnaire.

6. Quand demander de l'aide

  • Tu vois clairement le portefeuille d'un mauvais coach.
  • Une mission est incomprehensible ou technique.
  • Un bouton principal bloque ton travail.
  • Les chiffres de check-ups, rebooking ou clients semblent impossibles.
